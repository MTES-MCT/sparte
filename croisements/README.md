# Croisements possibles : Metadata INSEE x Consommation d'espaces NAF

## Contrainte temporelle

- **Consommation (MAJIC/Fichiers fonciers)** : flux annuels (chaque annee de ~2009 a ~2023)
- **Recensement INSEE** : 3 snapshots - P11/C11 (~2010), P16/C16 (~2015), P22/C22 (~2022)
- **Salaires (DADS/DSN)** : 1 snapshot (2023)
- **Etablissements (SIRENE)** : stock fin 2023, creations annuelles 2014-2023
- **Tourisme** : 1 snapshot (2026)
- **Electeurs** : 4 snapshots (2020, 2021, 2022, 2024)
- **BPE (equipements)** : 1 snapshot (2024)

**Approche recommandee** : Pour les donnees recensement, calculer des *variations entre periodes* (delta P22-P16, delta P16-P11) et les croiser avec la *consommation cumulee sur les memes periodes* (somme des flux annuels 2011-2016, 2016-2022).
